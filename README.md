# Te.Co Pandawa POS v3.0.0

Versi perbaikan dengan laporan owner harian, mingguan, dan bulanan; pemetaan resep Coffee Milo dan Milo Malaysia; pilihan bulan export; penyimpanan transaksi yang tahan login ulang; sinkronisasi merge-safe Firebase; serta backup cloud harian dan manual.

Buka `index.html` melalui GitHub Pages.

Konfigurasi keamanan dan aktivasi cloud terdapat di `FIREBASE_SETUP.md` serta `database.rules.json`.
